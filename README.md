# InterviewAI — AI Mock Interviewer

> © 2025 Navesh R. All rights reserved.

An AI-powered mock interview platform. Practice Technical + HR interviews for your dream role — powered by Claude AI.

---

## Features

- 🎯 **6 Target Roles** — Data Science, ML, Full Stack, AI Research, Data Analyst, Software Engineer
- 🤖 **Claude AI Interviewer** — Real, adaptive questions per role
- 🎤 **Voice + Text Input** — Speak or type your answers
- 📊 **Live Scoring** — Every answer scored 1-10 with feedback
- 🏆 **Results Dashboard** — Animated score, per-question feedback, overall assessment
- ⚡ **No backend** — Pure HTML/CSS/JS, works on GitHub Pages

---

## Setup

1. Open `index.html` in Chrome
2. Enter your Anthropic API key in the home screen
3. Select your role and start practicing!

---

## Deployment

### GitHub Pages
1. Create repo → push all 3 files
2. Settings → Pages → main → Save
3. Live at: `https://yourusername.github.io/repo-name`

---

## Tech Stack

- **Frontend** — HTML5, CSS3, Vanilla JS
- **AI** — Anthropic Claude API (claude-sonnet-4-6)
- **Voice** — Web Speech API (Chrome only)
- **Fonts** — Space Grotesk + Inter (Google Fonts)
